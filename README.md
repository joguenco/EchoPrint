# EchoPrint
Pos Print Server

REST API Server for print tickets in Point of Sale Printers

## Requirements
- Set as default POS printer (USB, Network, LPT, Serial, Wifi)

## Environment
- Free Pascal
- Lazarus IDE

## Web Client
- Vite
- pnpm

## Run Server
Compile and execute PrinterServer.exe or PrinterServer

## Run Client
```
pnpm i
```
```
pnpm run dev
```
